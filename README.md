# 🏋️ Bobby Iron Den

A simple fitness gym website built using HTML.  
This project is designed as a landing page for a gym/fitness brand called **Bobby Iron Den**.

---

## 📌 About the Project

**Bobby Iron Den** is a beginner-friendly web project that showcases a gym website layout.  
It includes a basic structure for a fitness brand website and can be expanded with CSS and JavaScript for more advanced features.

---

## 🚀 Features

- Home landing page
- Gym branding layout
- Basic structure for fitness content
- Lightweight and fast (pure HTML)
- Easy to customize and expand
